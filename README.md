# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
yarn install
```
3. Tener Nest CLi instalado
```
sudo npm i -g @nestjs/cli
```

4. Levatar base de datos
```

docker compose up -d
```
## Stack utilizado
  * MondoDb
  * Nest
